Projekt to responsywna strona internetowa (Laravel + Bootstrap), zawarte sa w nim nastepujace elementy:
- aplikacja JavaScript (Lightbox),
- funkcje na bazie Bootstrap + jQuery: dropdown-menu oraz parallax scrolling,
- utworzona baza danych na serwerze (wykorzystane migracje + Eloquent) oraz wyswietlanie zawartych w niej danych (system wysylania wiadomosci na serwer),
- uwierzytelnianie uzytkownika, dodatkowy profil administratora umozliwiajacy odczytywanie przeslanych na serwer wiadomosci,
- CMS (system zarzadzania trescia)- wprowadzona opcja zmiany profilowego zdjecia uzytkownika, wspolpracujaca z baza danych.



//ENG
The project contains a responsive website (Laravel + Bootstrap), with following functions:
- JavaScript application (Lightbox),
- Bootstrap + jQuery based functions: dropdown-menu and parallax scrolling,
- database on server (migrations + Eloquent) and displaying its content (ability to send messages),
- user authentication, administrator profile that allows reading sent messages,
- CMS- content management- ability to change profile avatar picture (also cooperating with database).
